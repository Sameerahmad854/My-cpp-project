# ATM Simulation
A simple **ATM simulation project*s* in C++ where a user can:

1. Check account balance
2. Deposit money
3. Withdraw money
4. Exit the system

## Features
1. User-friendly menu-based interface
2. Simple C++ console application
3. Uses basic input/output operations
4. Keeps track of account balance during program execution

## How to Run

1. Open terminal or command prompt.
2. Navigate to the project directory.
3. Compile the code using:

===bash===
g++ ATM_Simulation.cpp -o ATM_Simulation
