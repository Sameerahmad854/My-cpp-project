# Temperature Converter 

A simple **Temperature Converter** program in C++ that converts temperatures between Celsius and Fahrenheit.


## Features

1. Convert Celsius to Fahrenheit
2. Convert Fahrenheit to Celsius
3. User-friendly console-based interface
4. Beginner-friendly C++ project

---

## File Structure

TemperatureConverter/
1. main.cpp # Main C++ source code
2. README.md # Project documentation   

                    
## How to Compile & Run

### Step 1: Open Terminal or Command Prompt
Navigate to the project folder:


cd path/to/TemperatureConverter
## Step 2: Compile the Program
g++ main.cpp -o TemperatureConverter

## Step 3: Run the Executable
./TemperatureConverter

## Menu Options
1. Celsius to Fahrenheit
2. Fahrenheit to Celsius

## Notes

Input must be numeric.
Only basic conversions between Celsius and Fahrenheit are supported.
Console-based program with no persistent storage.
